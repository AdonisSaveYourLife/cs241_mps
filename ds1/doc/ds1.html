Instructions

To help you get started in C, we will be programming a very simple binary tree. To do this, you will be writing a simple library, so we have already provided you the code to run your MP. For example, test1.c contains:

btree_t tree;
btree_init(&tree);

btree_add(&tree, 3);
btree_add(&tree, 1);
btree_add(&tree, 2);
btree_add(&tree, 4);

btree_print(&tree);

In this MiniMP, you will need to implement:

  • The btree_entry_t data structure in btree.h: [see source]
  • The btree_init() function in btree.c: [documentation]
  • The btree_add() function btree.c: [documentation]
  • The btree_print_helper() helper function btree.c: [documentation]

Compile and Run

To compile, run:
make clean
make
To run the first test case:
./test1
To run the larger, second test case:
./test2
